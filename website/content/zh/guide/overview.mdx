## 什么是 Sharpside

**Sharpside** 是多平台预测市场跟单平台。核心做四件事：**接得广 / 找得到 / 跟得上 / 运营得动**。

当前 Phase 1 以 Polymarket 为唯一 Venue，通道 B（自托管 daemon）闭环已落地；通道 A（TG Deposit Wallet 委托代签）与 admin / 影子模式 / 风控在 Phase 1b 补齐中。

## 你能用它做什么

1. **发现** — 浏览 Venue 排行榜与交易者绩效（ROI、胜率、回撤、标签）。
2. **跟随** — 创建跟随关系，选择通道 A（TG）或通道 B（daemon）。
3. **执行** — 信号经 copier 下发；dry_run 可先跑通闭环再接真金。
4. **运营** — admin 审核映射/身份/热钥；影子模式对比第三方指标。
